Subtitle Component

UIKit SwiftUI New in v2.3.0

The Subtitle component provides secondary headings with customizable typography for creating clear visual hierarchies in your UI.

  • Overview
  • Properties
  • Methods
  • Examples

Overview

The Subtitle component was introduced in v2.3.0 to offer more flexible typography options separate from the Title component. Subtitles provide secondary headings that support:

  • Multiple styling presets for consistent typography
  • Custom font, color, alignment, and line spacing
  • Support for multiline text with truncation options
  • Accessibility features including Dynamic Type compatibility
  • Animation and transition effects

Use Subtitle components to create clear information hierarchies and to break content into manageable sections.

import SuperNovaStorybook

// Create a simple subtitle
let subtitle = SNSSubtitle(text: "Latest Updates")

// Add to your screen or view
screen.addComponent(subtitle)

Properties

Text Properties

Property Type Default Description
text String "" The text content displayed in the subtitle.
attributedText NSAttributedString? nil The attributed text content, which takes precedence over text if set.
font UIFont UIFont.systemFont(ofSize: 17, weight: .semibold) The font used for the subtitle text.
textColor UIColor .secondaryLabel The color of the subtitle text.
numberOfLines Int 0 (unlimited) The maximum number of lines to display. Set to 0 for unlimited lines.
lineBreakMode NSLineBreakMode .byTruncatingTail How text should be truncated when it doesn't fit within available lines.

Layout Properties

Property Type Default Description
textAlignment NSTextAlignment .natural The alignment of the subtitle text.
lineSpacing CGFloat 4 The spacing between lines of text.
letterSpacing CGFloat 0 The spacing between characters in the text.
padding UIEdgeInsets UIEdgeInsets(top: 0, left: 0, bottom: 8, right: 0) The padding around the subtitle text.
minimumScaleFactor CGFloat 0.75 The minimum scale factor for the text when using adjustsFontSizeToFitWidth.
adjustsFontSizeToFitWidth Bool false Whether the font size should be reduced to fit the text within available width.

Styling Properties

Property Type Default Description
style SNSSubtitleStyle .standard The preset style of the subtitle. Options include standard, prominent, muted, detail, caption.
isSelectable Bool false Whether the subtitle text can be selected by the user.
isInteractive Bool false Whether the subtitle responds to tap interactions.
underlineStyle NSUnderlineStyle? nil The style of underline to apply to the text, if any.

Accessibility Properties

Property Type Default Description
adjustsFontForContentSizeCategory Bool true Whether the font size adjusts based on the user's preferred content size category.
accessibilityLabel String? nil A custom accessibility label for VoiceOver. If nil, the subtitle text is used.
accessibilityTraits UIAccessibilityTraits .header The accessibility traits of the subtitle.

Text Handling Methods

Method Parameters Return Type Description
setText(_:) text: String Void Sets the subtitle's text content.
setAttributedText(_:) attributedText: NSAttributedString Void Sets the subtitle's attributed text content.
appendText(_:) text: String Void Appends text to the existing content.
applyMarkdown(_:) markdown: String Void Renders basic markdown formatting for the text.

Styling Methods

Method Parameters Return Type Description
applyStyle(_:) style: SNSSubtitleStyle Void Applies a predefined style to the subtitle.
setFont(_:adjustsForContentSizeCategory:) font: UIFont, adjustsForContentSizeCategory: Bool = true Void Sets the font with option for Dynamic Type support.
setTextAppearance(font:textColor:alignment:) font: UIFont, textColor: UIColor, alignment: NSTextAlignment Void Sets multiple text appearance properties at once.
setTextAttributes(_:) attributes: [NSAttributedString.Key: Any] Void Sets custom text attributes for the subtitle.
addUnderline(style:color:) style: NSUnderlineStyle = .single, color: UIColor? = nil Void Adds an underline to the subtitle text.

Animation Methods

Method Parameters Return Type Description
animateTextChange(to:duration:completion:) text: String, duration: TimeInterval = 0.3, completion: (() -> Void)? = nil Void Animates changing the subtitle text with a fade transition.
fadeIn(duration:completion:) duration: TimeInterval = 0.3, completion: (() -> Void)? = nil Void Fades in the subtitle from transparent to fully visible.
fadeOut(duration:completion:) duration: TimeInterval = 0.3, completion: (() -> Void)? = nil Void Fades out the subtitle from visible to transparent.

Examples

Basic Subtitle Usage

import SuperNovaStorybook

// Create a standard subtitle
let subtitle = SNSSubtitle(text: "Latest Products")

// Add to screen or view
screen.addComponent(subtitle)

Using Different Subtitle Styles

import SuperNovaStorybook

func createSubtitleExamples() {
    // Standard subtitle
    let standardSubtitle = SNSSubtitle(text: "Standard Subtitle", style: .standard)
    screen.addComponent(standardSubtitle)
    
    // Prominent subtitle - larger and bolder
    let prominentSubtitle = SNSSubtitle(text: "Prominent Subtitle", style: .prominent)
    screen.addComponent(prominentSubtitle)
    
    // Muted subtitle - less emphasis
    let mutedSubtitle = SNSSubtitle(text: "Muted Subtitle", style: .muted)
    screen.addComponent(mutedSubtitle)
    
    // Detail subtitle - smaller text
    let detailSubtitle = SNSSubtitle(text: "Detail Subtitle", style: .detail)
    screen.addComponent(detailSubtitle)
    
    // Caption subtitle - smallest text
    let captionSubtitle = SNSSubtitle(text: "Caption Subtitle", style: .caption)
    screen.addComponent(captionSubtitle)
}

Custom Styling

import SuperNovaStorybook

func createCustomStyledSubtitle() -> SNSSubtitle {
    // Create a subtitle with custom styling
    let subtitle = SNSSubtitle(text: "Upcoming Events")
    
    // Customize appearance
    subtitle.font = UIFont.systemFont(ofSize: 18, weight: .medium)
    subtitle.textColor = .systemIndigo
    subtitle.textAlignment = .center
    subtitle.lineSpacing = 6
    subtitle.letterSpacing = 0.5
    subtitle.padding = UIEdgeInsets(top: 4, left: 0, bottom: 12, right: 0)
    
    // Add an underline
    subtitle.addUnderline(style: .single, color: .systemIndigo)
    
    return subtitle
}

Attributed Text with Markdown

import SuperNovaStorybook

func createMarkdownSubtitle() -> SNSSubtitle {
    // Create a subtitle with markdown formatting
    let subtitle = SNSSubtitle(text: "")
    
    // Apply markdown formatting
    subtitle.applyMarkdown("Explore our *premium* selection with **exclusive** benefits")
    
    return subtitle
}

Interactive Subtitle

import SuperNovaStorybook

func createInteractiveSubtitle() -> SNSSubtitle {
    // Create a subtitle that can be tapped
    let subtitle = SNSSubtitle(text: "View All Categories")
    
    // Make it interactive
    subtitle.isInteractive = true
    subtitle.textColor = .systemBlue
    
    // Add tap gesture handler
    subtitle.onTap = {
        print("Subtitle was tapped!")
        // Navigate to categories screen or perform other actions
    }
    
    return subtitle
}

SwiftUI Integration

import SwiftUI
import SuperNovaStorybook

struct SubtitleDemoView: View {
    var body: some View {
        VStack(alignment: .leading, spacing: 20) {
            // Use subtitle component in SwiftUI
            SNSSubtitleRepresentable {
                let subtitle = SNSSubtitle(text: "Featured Articles")
                subtitle.style = .prominent
                return subtitle
            }
            .frame(height: 30)
            
            // Regular SwiftUI content
            Text("This demonstrates how to use SuperNovaStorybook components within SwiftUI views.")
                .font(.body)
                .foregroundColor(.secondary)
        }
        .padding()
    }
}
