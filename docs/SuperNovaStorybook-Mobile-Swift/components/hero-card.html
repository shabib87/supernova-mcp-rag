Hero Card Component

UIKit SwiftUI New in v2.3.0

The Hero Card component is designed for high-impact, visually appealing content display, typically used in featured sections or for highlighting important information with an accompanying image.

  • Overview
  • Properties
  • Methods
  • Examples

Overview

The Hero Card component provides a prominent way to display content with visual emphasis. It's commonly used at the top of screens, feeds, or as a featured element. Hero Cards include:

  • Large feature image that can be positioned in various ways
  • Title and subtitle text with customizable styling
  • Optional call-to-action buttons
  • Customizable overlay effects and gradients
  • Support for custom aspect ratios and image handling

Hero Cards are designed to be eye-catching and provide clear information with strong visual support.

import SuperNovaStorybook

// Create a basic hero card
let heroCard = SNSHeroCard(
    title: "Summer Collection",
    subtitle: "Explore our new seasonal items",
    image: UIImage(named: "summer_hero")
)

// Add to your screen
screen.addComponent(heroCard)

Properties

Content Properties

Property Type Default Description
title String "" The main title text of the hero card.
subtitle String "" The secondary subtitle text displayed beneath the title.
image UIImage? nil The feature image displayed in the hero card.
imageURL URL? nil URL for remote image loading. Will be used if image is nil.
placeholderImage UIImage? nil Image displayed while the main image is loading from URL.
badgeText String? nil Optional badge text displayed in the corner of the card (e.g., "New", "Featured").

Styling Properties

Property Type Default Description
cardStyle SNSHeroCardStyle .imageBackground The overall style of the hero card. Options include imageBackground, imageSide, imageTop, imageBottom.
aspectRatio CGFloat 1.78 (16:9) The width to height ratio of the card.
cornerRadius CGFloat 12 The corner radius of the card.
overlayColor UIColor UIColor.black.withAlphaComponent(0.3) Color overlay applied on top of the image to improve text legibility.
useGradientOverlay Bool true Whether to use a gradient overlay instead of a solid overlay color.
titleColor UIColor .white The color of the title text.
subtitleColor UIColor .white The color of the subtitle text.
titleFont UIFont UIFont.systemFont(ofSize: 20, weight: .bold) The font used for the title.
subtitleFont UIFont UIFont.systemFont(ofSize: 16, weight: .regular) The font used for the subtitle.
contentInsets UIEdgeInsets UIEdgeInsets(top: 16, left: 16, bottom: 16, right: 16) Insets for the text content relative to the card edges.

Interaction Properties

Property Type Default Description
isPressable Bool true Whether the card responds to touch with visual feedback.
pressAnimationDuration TimeInterval 0.1 Duration of the press animation in seconds.
pressAnimationScale CGFloat 0.97 Scale factor when the card is pressed.
onPress (() -> Void)? nil Callback function executed when the card is pressed.

Methods

Initialization

// Basic initializer with title and image
init(title: String, image: UIImage? = nil)

// Full initializer
init(title: String, subtitle: String? = nil, image: UIImage? = nil)

// Initialize with remote image URL
init(title: String, subtitle: String? = nil, imageURL: URL, placeholderImage: UIImage? = nil)

// Initialize with style
init(title: String, subtitle: String? = nil, image: UIImage? = nil, style: SNSHeroCardStyle)

Content Methods

Method Parameters Return Type Description
setImage(_:contentMode:) image: UIImage, contentMode: UIView.ContentMode = .scaleAspectFill Void Sets the card's image and content mode.
loadImage(from:placeholder:) url: URL, placeholder: UIImage? = nil Void Loads an image from a URL with optional placeholder.
addBadge(text:color:) text: String, color: UIColor = .systemRed Void Adds a badge with text to the top corner of the card.
addButton(title:style:action:) title: String, style: SNSButtonStyle = .primary, action: @escaping () -> Void SNSButton Adds a button to the card and returns the button.

Styling Methods

Method Parameters Return Type Description
applyStyle(_:) style: SNSHeroCardStyle Void Applies a preset style to the card.
setOverlay(color:) color: UIColor Void Sets a solid color overlay on the image.
setGradientOverlay(colors:locations:) colors: [UIColor], locations: [NSNumber]? = nil Void Sets a gradient overlay on the image with the given colors and locations.
setTitleStyle(font:color:) font: UIFont, color: UIColor Void Sets the font and color for the title text.
setSubtitleStyle(font:color:) font: UIFont, color: UIColor Void Sets the font and color for the subtitle text.

Examples

Basic Hero Card

import SuperNovaStorybook

// Create a basic hero card with image
let heroCard = SNSHeroCard(
    title: "Summer Collection",
    subtitle: "Explore our new seasonal items",
    image: UIImage(named: "summer_hero")
)

// Customize corner radius and add to screen
heroCard.cornerRadius = 16
screen.addComponent(heroCard)

Hero Card with Remote Image

import SuperNovaStorybook

func createRemoteImageHeroCard() -> SNSHeroCard {
    // Create URL for remote image
    let imageURL = URL(string: "https://example.com/path/to/image.jpg")!
    
    // Create hero card with remote image URL
    let heroCard = SNSHeroCard(
        title: "Breaking News",
        subtitle: "The latest updates from around the world",
        imageURL: imageURL,
        placeholderImage: UIImage(named: "news_placeholder")
    )
    
    // Add a badge
    heroCard.addBadge(text: "LIVE", color: .systemRed)
    
    // Set action for when the card is tapped
    heroCard.onPress = {
        print("Hero card was tapped!")
    }
    
    return heroCard
}

Styled Hero Card with Button

import SuperNovaStorybook

func createStyledHeroCard() -> SNSHeroCard {
    // Create hero card with side image style
    let heroCard = SNSHeroCard(
        title: "Premium Membership",
        subtitle: "Get exclusive access to premium content and features",
        image: UIImage(named: "premium_banner"),
        style: .imageSide
    )
    
    // Customize appearance
    heroCard.aspectRatio = 2.0 // 2:1 aspect ratio
    heroCard.useGradientOverlay = true
    heroCard.setGradientOverlay(
        colors: [
            UIColor.black.withAlphaComponent(0.7),
            UIColor.black.withAlphaComponent(0.3)
        ],
        locations: [0.0, 1.0]
    )
    
    // Style the text
    heroCard.setTitleStyle(
        font: UIFont.systemFont(ofSize: 24, weight: .bold),
        color: .white
    )
    heroCard.setSubtitleStyle(
        font: UIFont.systemFont(ofSize: 16, weight: .regular),
        color: UIColor.white.withAlphaComponent(0.8)
    )
    
    // Add a call-to-action button
    let button = heroCard.addButton(title: "Upgrade Now", style: .primary) {
        print("Upgrade button tapped!")
    }
    button.cornerRadius = 8
    
    return heroCard
}

Interactive Card with SwiftUI

import SwiftUI
import SuperNovaStorybook

struct HeroCardView: View {
    @State private var showDetails = false
    
    var body: some View {
        VStack {
            // Create a SwiftUI-compatible hero card
            SNSHeroCardRepresentable {
                let heroCard = SNSHeroCard(
                    title: "SwiftUI Integration",
                    subtitle: "Using SuperNovaStorybook components in SwiftUI",
                    image: UIImage(named: "swiftui_hero")
                )
                
                heroCard.onPress = {
                    // Update SwiftUI state when card is pressed
                    self.showDetails.toggle()
                }
                
                return heroCard
            }
            .frame(height: 200)
            .padding(.horizontal)
            
            // Show additional content when card is tapped
            if showDetails {
                Text("Additional details appear here when the card is tapped.")
                    .padding()
                    .background(Color(.secondarySystemBackground))
                    .cornerRadius(8)
                    .padding()
                    .transition(.opacity)
                    .animation(.easeInOut)
            }
        }
    }
}
