View Component

UIKit SwiftUI

The View component is a versatile container for organizing and displaying content within your application. It provides flexible layout options and can contain other components.

  • Overview
  • Properties
  • Methods
  • Examples

Overview

The View component serves as a flexible container in the SuperNovaStorybook component hierarchy. Unlike the Screen component, which acts as the root container, View components can be nested to create complex layouts. Views have the following features:

  • Flexible layout options (vertical, horizontal, grid)
  • Built-in padding and margin controls
  • Custom background and border styling
  • Shadow effects and rounded corners
  • Animations and transitions

Views are ideal for grouping related components together and creating reusable UI sections.

import SuperNovaStorybook

// Create a simple card view
let cardView = SNSView(style: .card)
cardView.padding = UIEdgeInsets(top: 16, left: 16, bottom: 16, right: 16)

// Add content to the card
let title = SNSTitle(text: "Important Information")
cardView.addComponent(title)

let body = SNSBody(text: "This is some important information...")
cardView.addComponent(body)

// Add the card to a screen or another view
screen.addComponent(cardView)

Properties

Public Properties

Property Type Default Description
backgroundColor UIColor .clear The background color of the view.
style SNSViewStyle .plain The preset style of the view. Options include plain, card, grouped, inset.
padding UIEdgeInsets UIEdgeInsets.zero The internal padding for the view's content.
margin UIEdgeInsets UIEdgeInsets.zero The external margin around the view.
cornerRadius CGFloat 0 The corner radius of the view.
borderWidth CGFloat 0 The width of the view's border.
borderColor UIColor .clear The color of the view's border.
shadowOpacity Float 0 The opacity of the view's shadow (0.0 - 1.0).
shadowRadius CGFloat 0 The blur radius of the view's shadow.
shadowOffset CGSize CGSize.zero The offset of the view's shadow.
shadowColor UIColor .black The color of the view's shadow.
components [SNSComponent] [] An array of components contained in this view.

Layout Properties

Property Type Default Description
layoutMode SNSLayoutMode .vertical The layout direction for components. Options are vertical, horizontal, grid.
spacing CGFloat 8 The spacing between components in the view.
alignment SNSAlignment .leading The alignment of components within the view. Options include leading, trailing, center, fill.
isScrollEnabled Bool false Determines if the view's content can be scrolled.
heightMode SNSHeightMode .wrap Controls how the view's height is determined. Options are wrap, fixed, proportional.
height CGFloat 0 The fixed height when heightMode is set to .fixed or the height proportion when set to .proportional.

Methods

Initialization

// Default initializer
init()

// Initialize with style
init(style: SNSViewStyle)

// Initialize with layout mode
init(layoutMode: SNSLayoutMode, spacing: CGFloat = 8)

// Initialize with full configuration
init(style: SNSViewStyle, layoutMode: SNSLayoutMode, spacing: CGFloat = 8)

Component Management

Method Parameters Return Type Description
addComponent(_:) component: SNSComponent Void Adds a component to the view.
insertComponent(_:at:) component: SNSComponent, index: Int Void Inserts a component at the specified index.
removeComponent(_:) component: SNSComponent Bool Removes a component from the view. Returns true if the component was found and removed.
removeAllComponents() None Void Removes all components from the view.
getComponent(withId:) id: String SNSComponent? Returns the component with the specified ID, if it exists.

Styling Methods

Method Parameters Return Type Description
applyStyle(_:) style: SNSViewStyle Void Applies a preset style to the view.
applyCardStyle() None Void Applies the card style to the view with default settings.
applyCardStyle(cornerRadius:shadowOpacity:) cornerRadius: CGFloat, shadowOpacity: Float Void Applies the card style with custom corner radius and shadow opacity.
setBorder(width:color:) width: CGFloat, color: UIColor Void Sets the border width and color for the view.
setShadow(opacity:radius:offset:color:) opacity: Float, radius: CGFloat, offset: CGSize, color: UIColor Void Sets the shadow properties for the view.

Examples

Card View

import SuperNovaStorybook

func createCardView() -> SNSView {
    // Create a card-style view
    let cardView = SNSView(style: .card)
    
    // Customize the card appearance
    cardView.padding = UIEdgeInsets(top: 16, left: 16, bottom: 16, right: 16)
    cardView.margin = UIEdgeInsets(top: 8, left: 16, bottom: 8, right: 16)
    cardView.cornerRadius = 12
    
    // Add content to the card
    let title = SNSTitle(text: "Daily Summary")
    cardView.addComponent(title)
    
    let body = SNSBody(text: "Your daily summary shows your activity progress...")
    cardView.addComponent(body)
    
    // Add a divider
    let divider = SNSDivider()
    cardView.addComponent(divider)
    
    // Add footer content
    let footerText = SNSBody(text: "Tap to view more details")
    footerText.textColor = .secondaryLabel
    footerText.fontSize = 14
    cardView.addComponent(footerText)
    
    return cardView
}

Horizontal Layout

import SuperNovaStorybook

func createHorizontalLayout() -> SNSView {
    // Create a view with horizontal layout
    let horizontalView = SNSView(layoutMode: .horizontal, spacing: 12)
    horizontalView.alignment = .center
    
    // Add an icon
    let iconView = SNSImageView(image: UIImage(systemName: "star.fill"))
    iconView.tintColor = .systemYellow
    iconView.width = 32
    iconView.height = 32
    horizontalView.addComponent(iconView)
    
    // Create a nested vertical view for text
    let textContainer = SNSView(layoutMode: .vertical, spacing: 4)
    
    // Add title and subtitle to the text container
    let title = SNSTitle(text: "Featured Item")
    title.fontSize = 16
    textContainer.addComponent(title)
    
    let subtitle = SNSSubtitle(text: "Special promotion available")
    subtitle.fontSize = 14
    textContainer.addComponent(subtitle)
    
    // Add the text container to the horizontal view
    horizontalView.addComponent(textContainer)
    
    return horizontalView
}

Grid Layout

import SuperNovaStorybook

func createGridLayout() -> SNSView {
    // Create a view with grid layout
    let gridView = SNSView(layoutMode: .grid, spacing: 16)
    gridView.gridColumns = 2
    
    // Add multiple items to the grid
    for i in 1...4 {
        // Create a card for each grid item
        let card = SNSView(style: .card)
        card.padding = UIEdgeInsets(top: 12, left: 12, bottom: 12, right: 12)
        card.cornerRadius = 8
        
        // Add content to the card
        let title = SNSTitle(text: "Item \(i)")
        title.fontSize = 16
        card.addComponent(title)
        
        let body = SNSBody(text: "This is grid item number \(i)")
        body.fontSize = 14
        card.addComponent(body)
        
        // Add the card to the grid
        gridView.addComponent(card)
    }
    
    return gridView
}

Nested Views with Custom Styling

import SuperNovaStorybook

func createNestedViews() -> SNSView {
    // Create the main container
    let containerView = SNSView()
    containerView.backgroundColor = .systemBackground
    containerView.padding = UIEdgeInsets(top: 16, left: 16, bottom: 16, right: 16)
    
    // Add a title
    let sectionTitle = SNSTitle(text: "Settings")
    containerView.addComponent(sectionTitle)
    
    // Create a grouped settings view
    let settingsView = SNSView(style: .grouped)
    settingsView.cornerRadius = 10
    settingsView.backgroundColor = .secondarySystemBackground
    settingsView.margin = UIEdgeInsets(top: 16, left: 0, bottom: 0, right: 0)
    
    // Add setting items as rows
    for (index, title) in ["Profile", "Notifications", "Privacy", "Help"].enumerated() {
        // Create a row view for each setting
        let rowView = SNSView(layoutMode: .horizontal, spacing: 12)
        rowView.padding = UIEdgeInsets(top: 12, left: 16, bottom: 12, right: 16)
        rowView.alignment = .center
        
        // Add setting title
        let titleLabel = SNSBody(text: title)
        titleLabel.fontSize = 16
        rowView.addComponent(titleLabel)
        
        // Add spacer to push the chevron to the right
        let spacer = SNSSpacer()
        rowView.addComponent(spacer)
        
        // Add chevron icon
        let chevron = SNSImageView(image: UIImage(systemName: "chevron.right"))
        chevron.tintColor = .systemGray3
        chevron.width = 16
        chevron.height = 16
        rowView.addComponent(chevron)
        
        // Add the row to the settings view
        settingsView.addComponent(rowView)
        
        // Add divider between rows (except for the last one)
        if index < 3 {
            let divider = SNSDivider()
            divider.margin = UIEdgeInsets(left: 16, right: 16)
            settingsView.addComponent(divider)
        }
    }
    
    // Add the settings view to the container
    containerView.addComponent(settingsView)
    
    return containerView
}
