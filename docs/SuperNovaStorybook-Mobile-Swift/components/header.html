Header Component

UIKit SwiftUI

The Header component provides navigation, branding, and context at the top of your screens with support for titles, buttons, and search functionality.

  • Overview
  • Properties
  • Methods
  • Examples

Overview

The Header component serves as the top navigation and contextual element in your application. It provides various features including:

  • Multiple style variants for different use cases
  • Support for title, subtitle, and accessory views
  • Built-in navigation controls (back button, close button)
  • Search capability with integrated search bar
  • Safe area handling and scroll behavior options

Headers are commonly used to provide context about the current view, navigation controls, and access to primary actions.

import SuperNovaStorybook

// Create a simple header with a title
let header = SNSHeader(title: "Home")

// Add to your screen
screen.addComponent(header)

Properties

Content Properties

Property Type Default Description
title String "" The main title displayed in the header.
subtitle String? nil Optional subtitle displayed below the main title.
attributedTitle NSAttributedString? nil Attributed string for the title, taking precedence over title if set.
attributedSubtitle NSAttributedString? nil Attributed string for the subtitle, taking precedence over subtitle if set.
titleView UIView? nil Custom view to display in place of the title and subtitle.
leftItems [SNSHeaderItem] [] Array of items displayed on the left side of the header.
rightItems [SNSHeaderItem] [] Array of items displayed on the right side of the header.
logo UIImage? nil Optional logo image to display in the header.

Appearance Properties

Property Type Default Description
style SNSHeaderStyle .standard The visual style of the header. Options include standard, large, compact, transparent, prominent.
backgroundColor UIColor .systemBackground The background color of the header.
titleColor UIColor .label The color of the title text.
subtitleColor UIColor .secondaryLabel The color of the subtitle text.
titleFont UIFont UIFont.boldSystemFont(ofSize: 17) The font used for the title.
subtitleFont UIFont UIFont.systemFont(ofSize: 13) The font used for the subtitle.
titleAlignment NSTextAlignment .center The alignment of the title text.
blurEffect UIBlurEffect.Style? nil The blur effect style applied to the header background. If set, overrides backgroundColor.
isTranslucent Bool false Whether the header has a translucent background.
borderStyle SNSHeaderBorderStyle .bottomDivider The border style for the header. Options include none, bottomDivider, shadow.
dividerColor UIColor .separator The color of the bottom divider when borderStyle is .bottomDivider.
shadowOpacity Float 0.2 The opacity of the shadow when borderStyle is .shadow.

Layout Properties

Property Type Default Description
height CGFloat 44.0 The height of the header content area (excluding safe area insets).
largeHeaderHeight CGFloat 96.0 The height of the header when style is .large.
contentInsets UIEdgeInsets UIEdgeInsets(top: 0, left: 16, bottom: 0, right: 16) The insets for the header's content relative to its edges.
respectsSafeArea Bool true Whether the header respects the safe area insets at the top.
scrollBehavior SNSHeaderScrollBehavior .fixed How the header responds to scrolling. Options include fixed, hidingOnScroll, condensingOnScroll, pinToTop.

Navigation Properties

Property Type Default Description
showsBackButton Bool false Whether to show a back button on the left side.
backButtonTitle String? nil Optional title for the back button. If nil, uses the system default.
backButtonIcon UIImage? nil Custom icon for the back button. If nil, uses the system default.
showsCloseButton Bool false Whether to show a close button on the left side.
backButtonHandler (() -> Void)? nil Handler called when the back button is tapped.
closeButtonHandler (() -> Void)? nil Handler called when the close button is tapped.

Search Properties

Property Type Default Description
showsSearchBar Bool false Whether to show a search bar within the header.
searchBarPlaceholder String "Search" The placeholder text for the search bar.
searchBarPosition SNSHeaderSearchPosition .belowTitle The position of the search bar. Options include belowTitle, inlineWithTitle, replacing.

Examples

Standard Header with Title

import SuperNovaStorybook

func createStandardHeader() -> SNSHeader {
    // Create a standard header with title
    let header = SNSHeader(title: "Profile")
    
    // Add a back button
    header.setShowsBackButton(true, withTitle: "Home")
    header.setBackButtonHandler {
        print("Back button tapped")
        // Navigate back to previous screen
    }
    
    // Add a right button
    header.addRightItem(title: "Edit") {
        print("Edit button tapped")
        // Show edit mode
    }
    
    return header
}

Large Style Header with Subtitle

import SuperNovaStorybook

func createLargeHeader() -> SNSHeader {
    // Create a large header with title and subtitle
    let header = SNSHeader(title: "Messages", style: .large)
    header.setSubtitle("12 unread conversations")
    
    // Customize appearance
    header.setTitleAppearance(
        font: UIFont.systemFont(ofSize: 28, weight: .bold),
        color: .label,
        alignment: .leading
    )
    header.setSubtitleAppearance(
        font: UIFont.systemFont(ofSize: 16, weight: .regular),
        color: .secondaryLabel
    )
    
    // Add right items
    header.addRightItem(icon: UIImage(systemName: "square.and.pencil")!) {
        print("Compose message")
    }
    
    header.addRightItem(icon: UIImage(systemName: "line.3.horizontal.decrease.circle")!) {
        print("Filter messages")
    }
    
    // Set scroll behavior
    header.setScrollBehavior(.condensingOnScroll)
    
    return header
}

Search Header

import SuperNovaStorybook

func createSearchHeader() -> SNSHeader {
    // Create a header with search functionality
    let header = SNSHeader(title: "Search")
    header.setShowsSearchBar(true, position: .belowTitle)
    header.setSearchBarPlaceholder("Search products...")
    
    // Handle search text changes
    header.setSearchTextDidChangeHandler { searchText in
        print("Search text changed: \(searchText)")
        // Update search results
    }
    
    // Handle search focus/blur
    header.setSearchDidBeginHandler {
        print("Search began")
        // Show recent searches or suggestions
    }
    
    header.setSearchDidEndHandler {
        print("Search ended")
        // Hide suggestions if needed
    }
    
    // Add a cancel button
    header.addRightItem(title: "Cancel") {
        print("Cancel search")
        header.dismissSearchBar()
        header.setSearchText("")
    }
    
    return header
}

Transparent Header for Media Content

import SuperNovaStorybook

func createTransparentHeader() -> SNSHeader {
    // Create a transparent header for use over images or videos
    let header = SNSHeader(style: .transparent)
    
    // Make it blend with the content below
    header.isTranslucent = true
    header.borderStyle = .none
    
    // Set up navigation with contrasting colors for visibility
    header.setShowsBackButton(true)
    header.setBackButtonIcon(UIImage(systemName: "chevron.left.circle.fill"))
    
    // Add right items that stand out against any background
    let saveItem = header.addRightItem(icon: UIImage(systemName: "bookmark.circle.fill")!) {
        print("Save item")
    }
    saveItem.tintColor = .white
    
    let shareItem = header.addRightItem(icon: UIImage(systemName: "square.and.arrow.up.circle.fill")!) {
        print("Share item")
    }
    shareItem.tintColor = .white
    
    return header
}

SwiftUI Integration

import SwiftUI
import SuperNovaStorybook

struct HeaderDemoView: View {
    @State private var isEditing = false
    @State private var searchText = ""
    
    var body: some View {
        VStack(spacing: 0) {
            // Header using SNSHeader in SwiftUI
            SNSHeaderRepresentable {
                let header = SNSHeader(title: "SwiftUI Demo", style: .standard)
                
                // Configure search
                header.setShowsSearchBar(true)
                header.setSearchBarPlaceholder("Search...")
                header.setSearchTextDidChangeHandler { [weak self] text in
                    // Update SwiftUI state
                    self?.searchText = text
                }
                
                // Add right item
                header.addRightItem(title: self.isEditing ? "Done" : "Edit") { [weak self] in
                    // Toggle edit state
                    self?.isEditing.toggle()
                }
                
                return header
            }
            .fixedSize(horizontal: false, vertical: true)
            
            // Content view
            ScrollView {
                VStack(alignment: .leading) {
                    if !searchText.isEmpty {
                        Text("Searching for: \(searchText)")
                            .padding()
                    }
                    
                    if isEditing {
                        Text("Edit mode is active")
                            .padding()
                            .background(Color.yellow.opacity(0.2))
                            .cornerRadius(8)
                            .padding()
                    }
                    
                    // Main content here...
                    ForEach(1...20, id: \.self) { i in
                        Text("Item \(i)")
                            .frame(maxWidth: .infinity, alignment: .leading)
                            .padding()
                            .background(Color(.secondarySystemBackground))
                            .cornerRadius(8)
                            .padding(.horizontal)
                    }
                }
                .padding(.vertical)
            }
        }
    }
}
