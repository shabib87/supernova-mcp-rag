Body Component

UIKit SwiftUI

The Body component is a versatile text container designed for main content display with rich formatting options and optimal readability.

  • Overview
  • Properties
  • Methods
  • Examples

Overview

The Body component is the primary component for displaying paragraphs of text content in your application. It's optimized for readability and offers extensive text formatting capabilities, including:

  • Rich text formatting with Markdown support
  • Multiple text styles for different content types
  • Dynamic Type support for accessibility
  • Custom typography and spacing control
  • Link detection and handling

Use Body components for article content, descriptions, instructions, and any other multi-paragraph text in your application.

import SuperNovaStorybook

// Create a simple body text component
let bodyText = SNSBody(text: "This is a paragraph of text that demonstrates the Body component. It supports multiple lines and automatically adjusts to fit the available width.")

// Add to your screen or view
screen.addComponent(bodyText)

Properties

Content Properties

Property Type Default Description
text String "" The plain text content displayed in the body.
attributedText NSAttributedString? nil Rich text content with styling attributes, takes precedence over text if set.
htmlContent String? nil HTML content to be rendered in the body text.
markdownContent String? nil Markdown content to be rendered in the body text.

Text Styling Properties

Property Type Default Description
style SNSBodyStyle .standard The text style preset. Options are standard, emphasized, small, quote, code.
font UIFont UIFont.systemFont(ofSize: 16) The font used for the body text.
textColor UIColor .label The color of the body text.
linkColor UIColor .systemBlue The color used for links within the text.
highlightColor UIColor? nil The background color for highlighted text portions.
fontSize CGFloat 16.0 The size of the font used for the body text.
fontWeight UIFont.Weight .regular The weight of the font used for the body text.

Layout Properties

Property Type Default Description
textAlignment NSTextAlignment .natural The alignment of the body text.
lineHeight CGFloat 1.4 The line height multiplier for the text.
paragraphSpacing CGFloat 8.0 The spacing between paragraphs in points.
firstLineHeadIndent CGFloat 0.0 The indentation of the first line of paragraphs.
headIndent CGFloat 0.0 The indentation of all lines except the first line.
padding UIEdgeInsets UIEdgeInsets(top: 0, left: 0, bottom: 16, right: 0) The padding around the body text.
numberOfLines Int 0 (unlimited) The maximum number of lines to display. Set to 0 for unlimited lines.
lineBreakMode NSLineBreakMode .byWordWrapping The line break mode when text exceeds available width.

Interaction Properties

Property Type Default Description
isSelectable Bool false Whether the text can be selected by the user.
isEditable Bool false Whether the text can be edited by the user.
dataDetectorTypes UIDataDetectorTypes .link Types of data to automatically detect and convert to links (URLs, phone numbers, etc.).
linkHandler ((URL) -> Bool)? nil Callback for handling link taps. Return true to indicate the link was handled.

Accessibility Properties

Property Type Default Description
adjustsFontForContentSizeCategory Bool true Whether the font size adjusts based on the user's preferred content size category.
accessibilityLabel String? nil A custom accessibility label. If nil, the text content is used.
accessibilityTraits UIAccessibilityTraits .staticText The accessibility traits of the body text.

Using Different Text Styles

import SuperNovaStorybook

func createBodyStyleExamples() {
    // Standard body style
    let standardBody = SNSBody(text: "This is the standard body text style.", style: .standard)
    screen.addComponent(standardBody)
    
    // Emphasized style (slightly larger and bolder)
    let emphasizedBody = SNSBody(text: "This text uses the emphasized style for important information.", style: .emphasized)
    screen.addComponent(emphasizedBody)
    
    // Small style (smaller text)
    let smallBody = SNSBody(text: "This text uses the small style for less important information or notes.", style: .small)
    screen.addComponent(smallBody)
    
    // Quote style (indented with a vertical line)
    let quoteBody = SNSBody(text: "This text demonstrates the quote style, which is useful for displaying quotations or excerpts from other sources.", style: .quote)
    screen.addComponent(quoteBody)
    
    // Code style (monospace font with background)
    let codeBody = SNSBody(text: "function example() { return 'Hello World'; }", style: .code)
    screen.addComponent(codeBody)
}

Rich Text with Markdown

import SuperNovaStorybook

func createMarkdownBodyText() -> SNSBody {
    // Create body text using markdown
    let markdown = """
    # Main Heading
    
    This is a paragraph with **bold text**, *italic text*, and `inline code`.
    
    ## Subheading
    
    - List item one
    - List item two
    - List item three
    
    [Visit our website](https://example.com)
    """
    
    let bodyText = SNSBody(markdown: markdown)
    return bodyText
}

Custom Styling with Links

import SuperNovaStorybook

func createCustomStyledBodyWithLinks() -> SNSBody {
    // Create a body component with custom styling
    let bodyText = SNSBody(text: "Learn more about our products and services.")
    
    // Apply custom styling
    bodyText.setFont(UIFont.systemFont(ofSize: 17, weight: .regular))
    bodyText.textColor = .darkGray
    bodyText.lineHeight = 1.5
    bodyText.paragraphSpacing = 12
    bodyText.padding = UIEdgeInsets(top: 0, left: 16, bottom: 16, right: 16)
    
    // Add a link
    bodyText.addLink(text: "products and services", url: URL(string: "https://example.com/products")!)
    
    // Handle link taps
    bodyText.setLinkHandler { url in
        print("Link tapped: \(url)")
        // Open the URL, navigate to a screen, etc.
        return true // Return true to indicate the link was handled
    }
    
    return bodyText
}

Advanced Text Formatting

import SuperNovaStorybook

func createAdvancedFormattedText() -> SNSBody {
    // Create attributed string with advanced formatting
    let paragraphStyle = NSMutableParagraphStyle()
    paragraphStyle.lineSpacing = 8
    paragraphStyle.paragraphSpacing = 16
    paragraphStyle.firstLineHeadIndent = 20
    
    let attributes: [NSAttributedString.Key: Any] = [
        .font: UIFont.systemFont(ofSize: 16),
        .foregroundColor: UIColor.darkGray,
        .paragraphStyle: paragraphStyle
    ]
    
    let attributedString = NSMutableAttributedString(
        string: "This is a demonstration of advanced text formatting capabilities of the Body component. You can create complex layouts and styles using attributed strings.",
        attributes: attributes
    )
    
    // Apply different formatting to specific ranges
    let boldRange = (attributedString.string as NSString).range(of: "advanced text formatting")
    attributedString.addAttributes(
        [.font: UIFont.boldSystemFont(ofSize: 16), .foregroundColor: UIColor.black],
        range: boldRange
    )
    
    let colorRange = (attributedString.string as NSString).range(of: "complex layouts and styles")
    attributedString.addAttributes(
        [.foregroundColor: UIColor.systemBlue, .underlineStyle: NSUnderlineStyle.single.rawValue],
        range: colorRange
    )
    
    // Create body with the attributed string
    let bodyText = SNSBody(attributedText: attributedString)
    return bodyText
}
