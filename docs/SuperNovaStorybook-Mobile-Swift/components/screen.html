Screen Component

UIKit SwiftUI

The Screen component is the root container for your application's UI. It serves as the main viewport and provides layout management for all child components.

  • Overview
  • Properties
  • Methods
  • Examples

Overview

The Screen component is the foundation of any SuperNovaStorybook application. It handles the following responsibilities:

  • Managing the root view hierarchy
  • Handling safe area insets
  • Providing responsive layout management
  • Supporting light and dark mode transitions
  • Coordinating navigation between views

Screens can contain Views, Headers, Footers, and other components to create a complete UI experience.

import SuperNovaStorybook

class MyViewController: UIViewController {
    override func viewDidLoad() {
        super.viewDidLoad()
        
        // Create a screen component
        let screen = SNSScreen()
        
        // Configure screen properties
        screen.backgroundColor = .systemBackground
        screen.safeAreaHandling = .respectAll
        
        // Add screen to view hierarchy
        view.addSubview(screen)
        
        // Add components to screen
        let header = SNSHeader(title: "Welcome")
        screen.addComponent(header)
    }
}

Properties

Public Properties

Property Type Default Description
backgroundColor UIColor .systemBackground The background color of the screen.
safeAreaHandling SNSSafeAreaHandling .respectAll Determines how the screen handles safe area insets. Options are respectAll, ignoreAll, respectTop, respectBottom.
isScrollEnabled Bool true Determines if the screen content can be scrolled.
navigationStyle SNSNavigationStyle .standard The navigation style of the screen. Options are standard, compact, minimal, custom.
components [SNSComponent] [] An array of components added to this screen.

Layout Properties

Property Type Default Description
contentInsets UIEdgeInsets UIEdgeInsets(top: 0, left: 16, bottom: 0, right: 16) The padding applied to the screen's content.
layoutMode SNSLayoutMode .vertical The layout direction for components. Options are vertical, horizontal, grid.
spacing CGFloat 16 The spacing between components in the screen.

Methods

Initialization

// Default initializer
init()

// Initialize with background color
init(backgroundColor: UIColor)

// Initialize with custom layout
init(layoutMode: SNSLayoutMode, spacing: CGFloat = 16)

Component Management

Method Parameters Return Type Description
addComponent(_:) component: SNSComponent Void Adds a component to the screen.
insertComponent(_:at:) component: SNSComponent, index: Int Void Inserts a component at the specified index.
removeComponent(_:) component: SNSComponent Bool Removes a component from the screen. Returns true if the component was found and removed.
removeAllComponents() None Void Removes all components from the screen.
getComponent(withId:) id: String SNSComponent? Returns the component with the specified ID, if it exists.

Navigation

Method Parameters Return Type Description
push(screen:animated:completion:) screen: SNSScreen, animated: Bool = true, completion: (() -> Void)? = nil Void Pushes a new screen onto the navigation stack.
pop(animated:completion:) animated: Bool = true, completion: (() -> Void)? = nil Void Pops the current screen from the navigation stack.
present(screen:animated:completion:) screen: SNSScreen, animated: Bool = true, completion: (() -> Void)? = nil Void Presents a screen modally.
dismiss(animated:completion:) animated: Bool = true, completion: (() -> Void)? = nil Void Dismisses the current modal screen.

Examples

Basic Screen Setup

import SuperNovaStorybook

class HomeViewController: UIViewController {
    override func viewDidLoad() {
        super.viewDidLoad()
        
        // Create a screen
        let screen = SNSScreen()
        screen.backgroundColor = .systemBackground
        view.addSubview(screen)
        screen.frame = view.bounds
        screen.autoresizingMask = [.flexibleWidth, .flexibleHeight]
        
        // Add a header
        let header = SNSHeader(title: "Home")
        screen.addComponent(header)
        
        // Add content
        let title = SNSTitle(text: "Welcome to the app!")
        screen.addComponent(title)
        
        let subtitle = SNSSubtitle(text: "Explore our features below")
        screen.addComponent(subtitle)
        
        let body = SNSBody(text: "Lorem ipsum dolor sit amet...")
        screen.addComponent(body)
        
        // Add a footer
        let footer = SNSFooter()
        footer.addButton(title: "Get Started", style: .primary) {
            print("Button tapped!")
        }
        screen.addComponent(footer)
    }
}

SwiftUI Integration

import SwiftUI
import SuperNovaStorybook

struct ContentView: View {
    var body: some View {
        SNSScreenRepresentable {
            let screen = SNSScreen()
            
            let header = SNSHeader(title: "SwiftUI Integration")
            screen.addComponent(header)
            
            let title = SNSTitle(text: "Using SuperNovaStorybook with SwiftUI")
            screen.addComponent(title)
            
            // Return the configured screen
            return screen
        }
        .edgesIgnoringSafeArea(.all)
    }
}

Custom Navigation

import SuperNovaStorybook

class NavigationExample {
    func setupNavigation() {
        let firstScreen = SNSScreen()
        firstScreen.backgroundColor = .systemBackground
        
        let header = SNSHeader(title: "First Screen")
        firstScreen.addComponent(header)
        
        let button = SNSButton(title: "Go to Second Screen", style: .primary) { [weak self] in
            self?.navigateToSecondScreen(from: firstScreen)
        }
        firstScreen.addComponent(button)
        
        // Present the first screen
        SNSNavigationController.shared.setRootScreen(firstScreen)
    }
    
    private func navigateToSecondScreen(from screen: SNSScreen) {
        let secondScreen = SNSScreen()
        secondScreen.backgroundColor = .systemBackground
        
        let header = SNSHeader(title: "Second Screen")
        header.showBackButton = true
        secondScreen.addComponent(header)
        
        // Push the second screen
        screen.push(screen: secondScreen, animated: true)
    }
}
