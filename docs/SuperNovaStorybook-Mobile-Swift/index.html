A powerful UI component library for Swift-based iOS applications

Introduction

SuperNovaStorybook is a comprehensive UI component library designed to help iOS developers build beautiful, consistent, and responsive mobile applications using Swift. Our components are designed with performance and accessibility in mind, ensuring your app looks great and works well for all users.

This documentation will guide you through the various components available in the library, their properties, methods, and usage examples.

import SuperNovaStorybook

// Initialize your UI with SuperNova components
let screen = SNSScreen()
let header = SNSHeader(title: "Welcome")
screen.addComponent(header)

Getting Started

Add SuperNovaStorybook to your project using Swift Package Manager:

dependencies: [
    .package(url: "https://github.com/supernova/storybook-mobile-swift.git", from: "2.3.0")
]

Or with CocoaPods:

pod 'SuperNovaStorybook', '~> 2.3.0'
